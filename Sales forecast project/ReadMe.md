# 📊 Sales Forecast Dashboard with Machine Learning

## 🚀 Project Overview

This project analyzes sales data using Power BI and predicts future sales using a Machine Learning model (Random Forest Regressor).

## 📌 Key Features

* Interactive Power BI Dashboard
* Business Insights (Region, Category, Seasonality)
* Sales Prediction using Machine Learning
* Comparison of Actual vs Predicted Sales

## 🧠 Machine Learning Model

* Algorithm: Random Forest Regressor
* Features Used: Month, Quantity, Profit, Region, Category, Segment
* Model Accuracy: R² ≈ 0.58

## 📊 Visualizations

* Sales Trends by Month
* Region-wise Performance
* Category Analysis
* Actual vs Predicted Sales

## 💡 Key Insights

* Strong seasonal peak in Nov–Dec
* West region performs best
* COD is dominant payment method
* Model captures overall trend with moderate accuracy

## 🛠️ Tools Used

* Power BI
* Python (Pandas, Scikit-learn)
* Jupyter Notebook

## 📂 Project Structure

* dataset/ → Raw data
* powerbi/ → Dashboard file
* ml/ → ML model notebook
* output/ → Predicted dataset

## 📈 Conclusion

The project demonstrates how data analysis and machine learning can be combined to generate business insights and forecast sales effectively.
